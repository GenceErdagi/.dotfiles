# Gence Dotfiles

My dotfiles repo. I use the gnu stow to mange this repo and it mainly revolves around the helix, yazi and zellij to get a similar experience to a ide environment.
Bevare that i im using custom versions of [helix](https://github.com/mattwparas/helix/tree/steel-event-system) to activate helix plugin system. If the pull request will get merged in main i will change my usage and streamline my process. Currently there are a lot of custom written and community written plugins that i am using in helix editor to improve it and connect the ide environment within the zellij, yazi, nushell and helix with minimal friction.
