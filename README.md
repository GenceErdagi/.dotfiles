# Gence Dotfiles

My dotfiles repo. I use the gnu stow to mange this repo and it mainly revolves around the helix, yazi and zellij to get a similar experience to a ide environment.
Bevare that i im using custom versions of [zellij](https://github.com/GenceErdagi/zellij/tree/go-to-swap-layout) and [helix](https://github.com/mattwparas/helix/tree/steel-event-system) to activate helix plugin system and custom written utility that handles swapping the swap layouts without flicker
